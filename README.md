### 🛠 Overview
This tool is designed for digital marketers and developers to audit GitHub repository authority. It calculates "Social Proof" scores based on star-to-fork ratios, follower engagement, and contribution consistency.

> **Note:** This is an internal tool used by the **GMBSells** operations team to optimize client profiles.

---

## ✨ Key Features
- **Profile Audit:** Analyzes the last 3 years of contribution data to ensure graph consistency.
- **Engagement Tracking:** Monitors the retention rate of repository stars and forks.
- **SEO Optimization:** Suggests keywords for repository topics to increase search visibility.
- **Auto-Graph:** Lightweight script to maintain daily activity pings.
